This project is a web-based community platform called ANGAT. The system includes pages for Home, Hanap Trabaho, Hanap Gawa, Negosyo, Puhunan, Training, user profile management, notifications, and contact support. Users can register, log in, post job, service, and negosyo listings, browse local opportunities, apply for jobs, send service requests, manage their own listings, and view training programs and funding information through a database-driven interface.

INSTRUCTIONS:

-RUNNING THE SYSTEM-

1 - OPEN `GROUP6_ANGAT.sln` IN VISUAL STUDIO

2 - RESTORE THE REQUIRED NUGET PACKAGES IF PROMPTED

3 - CONFIGURE THE DATABASE CONNECTION STRING IN `Web.config`

4 - MAKE SURE THE REQUIRED DATABASE TABLES AND VIEWS ARE ALREADY CREATED IN SQL SERVER OR AZURE SQL DATABASE

5 - PRESS `F5` OR CLICK `START` TO RUN THE WEB APPLICATION

-MAIN FEATURES-

1 - REGISTER A NEW ACCOUNT OR LOG IN USING AN EXISTING ACCOUNT

2 - BROWSE AVAILABLE JOB LISTINGS IN `HANAP TRABAHO`

3 - BROWSE AVAILABLE SERVICE LISTINGS IN `HANAP GAWA`

4 - VIEW LOCAL BUSINESS LISTINGS IN `NEGOSYO`

5 - READ FUNDING INFORMATION IN `PUHUNAN`

6 - VIEW TRAINING PROGRAMS IN `TRAINING`

7 - OPEN THE PROFILE PAGE TO MANAGE APPLICATIONS, REQUESTS, AND YOUR OWN LISTINGS

-POSTING FEATURES-

1 - LOG IN AS A USER TO POST A JOB LISTING

2 - LOG IN AS A USER TO POST A SERVICE LISTING

3 - LOG IN AS A USER TO REGISTER A NEGOSYO LISTING

4 - ADD DETAILS SUCH AS CATEGORY, BARANGAY, DESCRIPTION, TAGS, RATES, HOURS, AND CONTACT INFORMATION DEPENDING ON THE LISTING TYPE

-USER ACTIONS-

1 - APPLY TO JOB LISTINGS THROUGH THE `HANAP TRABAHO` PAGE

2 - SEND REQUESTS TO SERVICE LISTINGS THROUGH THE `HANAP GAWA` PAGE

3 - VIEW AND MANAGE ACTIVE OR ARCHIVED APPLICATIONS AND REQUESTS IN THE PROFILE PAGE

4 - EDIT OR DELETE YOUR OWN NEGOSYO LISTINGS

5 - UPDATE YOUR ACCOUNT DETAILS, PROFILE IMAGE, AND PASSWORD IN THE PROFILE PAGE

-OTHER FEATURES-

1 - NOTIFICATION SYSTEM FOR ACCOUNT AND LISTING-RELATED UPDATES

2 - DATABASE-DRIVEN DISPLAY OF JOBS, SERVICES, BUSINESSES, TRAINING PROGRAMS, AND FUNDING PROGRAMS

3 - FILTERING AND SORTING FOR SELECT PAGES

4 - CONTACT PAGE FOR USER INQUIRIES

5 - PROFILE PAGE TABS FOR APPLICATIONS, REQUESTS, JOB LISTINGS, SERVICE LISTINGS, AND NEGOSYO LISTINGS

-REQUIREMENTS-

1 - WINDOWS OPERATING SYSTEM

2 - VISUAL STUDIO WITH ASP.NET WEB FORMS SUPPORT

3 - .NET FRAMEWORK 4.8

4 - SQL SERVER OR AZURE SQL DATABASE

5 - INTERNET CONNECTION IF USING AZURE SQL OR HOSTED DEPLOYMENT
